# Smart Travel Planner

Smart Travel Planner is a microservice-based SOA project that helps users plan trips, manage destinations, dates and budgets, and receive recommendations for hotels, restaurants and attractions.

## Services

- Trip Service
- Recommendation Service
- API Gateway
- Discovery Server
- MCP Server

## Technologies

- Spring Boot
- PostgreSQL
- Docker Compose
- API Gateway
- Service Discovery
- Keycloak
- Kafka
- OpenFeign
- MCP Server
